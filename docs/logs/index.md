# Solvinter Logs

Logs are the living memory of Solvinter.

They document what was actually done before the work becomes docs, research or whitepapers.

Logs are chronological and practical. They should preserve decisions, experiments, equipment, costs, implementation notes and lessons learned while the work is still close to the field.
