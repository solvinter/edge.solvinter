# Git

## Purpose

Git is a distributed version control system.

It records every change made to files and allows projects to be synchronized across multiple computers.

## Why Solvinter uses Git

Every document, script and configuration is version controlled.

Git makes it possible to work from macOS, Fedora, Raspberry Pi and iPhone while maintaining a complete history of the project.

## Concepts

### Repository

A repository (repo) is a project together with its complete history.

### Commit

A commit is a snapshot of the repository at a specific point in time.

### Remote

A remote repository is a copy stored on another computer, such as GitHub.
